	## Projects

	I have worked on a few projects and these are one ones that I think are worthy of a mention here.

	- **[krunchpad](https://github.com/krunchiekrunch/krunchpad):**  A 12 key macropad powered by RP2040 with KMK firmware.
	- **[krunchpad-20](https://github.com/krunchiekrunch/krunchpad-20):** krunchpad but with 20 keys.
	- **[krunchboard-90](https://github.com/krunchiekrunch/krunchboard-90):** A 75% ANSI mechanical keyboard powered by a Raspberry Pi Pico with KMK firmware.
    - **[usb-hub-v2](https://github.com/krunchiekrunch/usb-hub-v2):** A 4 port USB hub that have both USB-A and USB-C connector for PC.
	- **[uk-crime-map](https://github.com/krunchiekrunch/uk-crime-map):** A tool to visualise and plot the location and details of crimes in the UK Police API.
	- **[GeoFS-Information-Display](https://github.com/krunchiekrunch/GeoFS-Information-Display):** A userscript extension for the web flight simulator GeoFS that displays various information about the flight.
	- **[bus-board](https://github.com/krunchiekrunch/bus-board):** A website to show information about local buses in the UK.
	- **[RPi-BBC-RSS-OLED](https://github.com/krunchiekrunch/RPi-BBC-RSS-OLED):** A program to display latest BBC news headlines on a 0.96" i2c display.
	- **[bme280-web](https://github.com/krunchiekrunch/bme280-web):** A self-hosted minimalistic website to display climate data from a BME280 sensor.
	- **[confessions-api](https://github.com/krunchiekrunch/confessions-api):** An API for sending, reading, rating, and commenting anonymous confessions.
	- **[JPEG-webcam-timelapse](https://github.com/krunchiekrunch/JPEG-webcam-timelapse):** A program to generate a timelapse GIF from JPEG stream webcams.

	## Radio and RF

	I have been interested in RF for quite a while, I remeber when I bought a [software-defined radio dongle](https://www.nooelec.com/store/sdr/sdr-receivers/nesdr/nesdr-mini.html) and started tracking aircrafts via ADS-B and Mode-S, I had so much fun just learning about radio frequencies. This wasn't something I thought I would be interested in but I am so glad I discovered it.

	I also learned about receving NOAA satellite images, particularlly the NOAA-15/18/19 satellites that transmits [APT](https://en.wikipedia.org/wiki/Automatic_picture_transmission) on 137MHz, I remember using a handheld radio and just recording the beeps of the APT signal, then later decoding them on SatDump.
